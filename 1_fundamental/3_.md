# JAVA

Java yaitu salah satu bahasa pemrograman populer yang dapat digunakan untuk mengembangkan aplikasi mobile, desktop, sampai website. Salah satu alasan populernya Java adalah fleksibilitasnya yang dapat digunakan di banyak platform.

## Instalasi Java
Yang pertama kita harus menginstal dulu JDK( Java Development Kit). 
Mengapa? JDK berfungsi untuk melakukan penulisan serta pengembangan program java. Kalau di C namanya include<stdio.h> sebagai library dalam penulisan.

 Instalasi untuk perangkat windows:
### 1. Download JDK
Menggunakan JDK 8 karena versi yang sudah lama dan stabil. Boleh mendownload yang versi lebih terbaru.
* klik link ini https://www.oracle.com/java/technologies/downloads/#java8-windows<br>
